// Трио MTF Synchrony

Три угла на один рынок.

Согласованное семейство из трёх мультитаймфреймовых осцилляторов, которые разделяют один визуальный язык, но отвечают на три разных вопроса. Используйте их рядом для полного чтения: где находится цена, насколько велико движение и кто побеждает.

Пять показаний RSI на таймфреймах с интервалами Фибоначчи, объединённых в одну взвешенную линию Master — синхронность моментума как визуальная радужная плотность. Отвечает: где в диапазоне?

  • ✓ Слияние RSI 5-TF (веса Фибоначчи)
  • ✓ Длины по TF 7/10/14/14/21 (база Уайлдера)
  • ✓ Таблица легенды 6×9 со столбцом состояния
  • ✓ Классическая дивергенция + динамический Black Swan
